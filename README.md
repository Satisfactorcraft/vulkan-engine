# Vulkan Engine

A high-performance graphics rendering engine built with **Vulkan** and **C++**.

## Overview

This project is a modern Vulkan-based graphics engine designed for efficient, low-level GPU access and rendering. Built with a focus on performance and flexibility, it provides a robust foundation for 3D graphics applications.

### Key Features

- **Modern Vulkan API** - Leverages the latest Vulkan graphics standard for optimal performance
- **C++ Implementation** - High-performance code written in modern C++ (94.1% of codebase)
- **Custom Shaders** - GLSL shaders for flexible rendering pipelines
- **Build System** - Makefile-based build configuration for cross-platform compilation

## Project Statistics

- **Primary Language**: C++ (94.1%)
- **Shaders**: GLSL (3.5%)
- **Build Configuration**: Makefile (1.6%)
- **Other**: 0.8%

## Getting Started

### Prerequisites

- C++17 or later
- Vulkan SDK (version 1.3+)
- CMake or Make build tools
- GPU with Vulkan support

### Building

```bash
# Clone the repository
git clone https://github.com/Satisfactorcraft/vulkan-engine.git
cd vulkan-engine

# Build using Make
make

# Or build with CMake (if supported)
mkdir build
cd build
cmake ..
make
```

### Architecture

- **C++** - Programming Language
- **GLSL** - Vertex and fragment Shader
- **CMake and Makefile** - Build System
